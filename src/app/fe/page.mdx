# 前端开发

前端主要负责用户界面的编写，在实际生产生活中，主要负责处理由服务端传来的数据并呈现给用户，或是收集用户输入的信息处理后递交给服务端。 {{ className: 'lead' }}

前端开发需要一定的审美，因为前端要经常对设计稿进行复原，有时还有类似于动画，乃至图形学渲染的需求。 {{ className: 'lead' }}

如果你喜欢 **直观地看到自己创造了什么** ，那你便适合前端的工作，因为前端几乎是软件开发里最容易看到成效的岗位。 {{ className: 'lead' }}
