# JS 基础语法

简单捋一捋 JS 的基础语法。{{ className: 'lead' }}

---

<Row>
  <Col>
    ## 声明变量 {{id: 'declare'}}

    使用关键字 `let` 声明变量，使用关键字 `function` 声明函数。在 JS 里，函数也是一种变量。

  </Col>
  <Col sticky>
    <CodeGroup title="JS 声明变量">
    ```js
    // 声明两个变量，JS 句尾绝大多数情况可以不加分号
    let a = 1
    let b = 2
    // 声明一个接受两个变量，返回其相加结果的函数 add
    function add(a, b) {
      return a + b
    }
    console.log(add(1, 2)) // 3
    ```
    </CodeGroup>
  </Col>
</Row>

    ## 输出结果 {{id: 'output'}}

<Row>
  <Col>
    ## 条件语句 {{id: 'condition'}}

    JS 的选择语句支持 **if else**、**switch case**、**三目表达式** 等写法。

  </Col>
  <Col sticky>
    <CodeGroup title="JS 中的 if 语句">
    ```js
    // 声明两个变量，JS 句尾绝大多数情况可以不加分号
    let a = 1
    if(a === 1) {
      console.log("a === 1")
    } else {
      console.log("a !== 1")
    }
    ```
    </CodeGroup>
    <CodeGroup title="JS 中的 switch 语句">
    ```js
    let a = 1
    switch (a) {
      case 1:
        console.log("a === 1")
        break
      default:
        console.log("a !== 1")
        break
    }
    ```
    </CodeGroup>
        <CodeGroup title="JS 中的三目表达式">
    ```js
    let a = 1
    a === 1 ? console.log("a === 1") : console.log("a !== 1")
    ```
    </CodeGroup>
  </Col>
</Row>

<Row>
  <Col>
    ## 循环语句 {{id: 'loop'}}

    JS 除了支持众多语言通用的 **while**、**for** 循环语句外，还支持例如 **for of** 以及其他众多的循环与遍历方式。

  </Col>
  <Col sticky>
    <CodeGroup title="JS 中的 while 循环">
    ```js
    // 声明两个变量，JS 句尾绝大多数情况可以不加分号
    let i = 0
    while(i < 10) {
      console.log(i)
      i += 1
    }
    ```
    </CodeGroup>
    <CodeGroup title="JS 中的 for 循环">
    ```js
    for(let i = 0; i < 10; i++ ) {
      console.log(i)
    }
    ```
    </CodeGroup>
        <CodeGroup title="JS 中的 for of 循环">
    ```js
    let arr = [1, 2, 3, 4, 5]
    // 在一些其他语言里，这个语法或许名为 for each
    for (let item of arr) {
      console.log(item) // item 为 arr 中的元素
    }
    ```
    </CodeGroup>
  </Col>
</Row>
