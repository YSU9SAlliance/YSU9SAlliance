    ## 声明变量 {{id: 'declare'}}

    使用关键字 `let` 声明变量，使用关键字 `function` 声明函数。在 JS 里，函数也是一种变量。

    在同一作用域内：
    - `let` 声明的变量，可以重复定义，但不能重复声明。**推荐使用**
    - `var` 声明的变量，可以重复声明与定义，但 **不推荐使用**
    - `const` 用于 **定义常量**，不可被重复声明与定义。

    - `function` 定义的函数，可以被重复声明与定义，后者覆盖前者，但不推荐重复声明与定义。

  </Col>
  <Col sticky>
    <CodeGroup title="JS 声明变量">
    ```js
    // 声明两个变量，JS 句尾绝大多数情况可以不加分号
    let x = 1
    let y = 2
    // 声明一个接受两个变量，返回其相加结果的函数 add
    function add(a, b) {
      return a + b
    }
    // 使用 const 声明，用箭头函数进行定义一个常量函数
    const minus = (a, b) => a - b

    console.log(add(x, y)) // 3
    console.log(minus(x, y)) // -1
    ```
    </CodeGroup>

  </Col>
</Row>

    ## 条件语句 {{id: 'condition'}}

    JS 的选择语句支持 **if else**、**switch case**、**三目表达式** 等写法。

{' '}

<Note>
  注意，其他语言的 **==** 与 **!=** 在 JS 中应该替换为 **===** 与 **!==**
</Note>

  </Col>
  <Col sticky>
    <CodeGroup title="JS 中的 if 语句">
    ```js
    // 声明两个变量，JS 句尾绝大多数情况可以不加分号
    let a = 1
    if(a === 1) {
      console.log("a === 1")
    } else {
      console.log("a !== 1")
    }
    ```
    </CodeGroup>
    <CodeGroup title="JS 中的 switch 语句">
    ```js
    let a = 1
    switch (a) {
      case 1:
        console.log("a === 1")
        break
      default:
        console.log("a !== 1")
        break
    }
    ```
    </CodeGroup>
        <CodeGroup title="JS 中的三目表达式">
    ```js
    let a = 1
    a === 1 ? console.log("a === 1") : console.log("a !== 1")
    ```
    </CodeGroup>
  </Col>
</Row>

    ## 循环语句 {{id: 'loop'}}

    JS 除了支持众多语言通用的 **while**、**for** 循环语句外，还支持例如 **for of** 以及其他众多的循环与遍历方式。

  </Col>
  <Col sticky>
    <CodeGroup title="JS 中的 while 循环">
    ```js
    let i = 0
    while(i < 10) {
      console.log(i)
      i += 1
    }
    ```
    </CodeGroup>
    <CodeGroup title="JS 中的 for 循环">
    ```js
    for(let i = 0; i < 10; i++ ) {
      console.log(i)
    }
    ```
    </CodeGroup>
        <CodeGroup title="JS 中的 for of 循环">
    ```js
    let arr = [1, 2, 3, 4, 5]
    // 在一些其他语言里，这个语法或许名为 for each
    for (let item of arr) {
      console.log(item) // item 为 arr 中的元素
    }
    ```
    </CodeGroup>
  </Col>
</Row>

    ## 函数 {{id: 'function'}}

    JS 除了通常的函数定义与调用，还支持匿名函数，一般使用 **箭头函数** 来定义匿名函数，箭头函数支持被重新命名

  </Col>
  <Col sticky>
    <CodeGroup title="JS 的函数定义与调用">
    ```js
    let a = 1
    let b = 2
    function add(a, b) {
      return a + b
    }
    console.log(add(a, b)) // 3
    ```
    </CodeGroup>

    <CodeGroup title="JS 中的箭头函数">
    ```js
    let a = 1
    let b = 2
    // 定义一个被命名的箭头函数
    const add = (a, b) => a + b
    console.log(add(a, b)) // 3
    // 直接使用匿名函数定义减法，并当场执行
    console.log(((a, b) => a - b)(a, b)) // -1
    ```
    </CodeGroup>

  </Col>
</Row>
