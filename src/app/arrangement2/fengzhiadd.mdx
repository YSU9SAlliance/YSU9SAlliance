# 如何从GitHub上拉取别人的项目代码，克隆到本地修改之后再上传到GitHub
Steps:
1. 本地新建一个文件夹
2. 将文件夹用VScode打开
3. 在VScode中打开控制栏(control+~)
4. 输入git clone URL，输入git init；输入git add . ;输入git commit -m "Initial commit";
5. 输入git branch 分支名称（一般以主要修改内容为名）（创建分支）
6. 输入git checkout 分支名称（切换到这一新建分支，后续修改将在这一分支上进行）
7. 修改
8. 点击右方图标Git
9. 点击+号
10. 起名: feat: 名称（feat：添加内容；fix：修改内容）（英文冒号）
11. 点击提交（暂存）
12. 点击同步更改
13. 回到GitHub找到源仓库
14. 点击pull request
15. 添加disruption
16. 点击提交
17. 等待仓库原主人回复（LGTM=已阅）
18. 得到肯定回复后直接点merge(注意点两次)
